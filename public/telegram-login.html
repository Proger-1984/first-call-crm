Как настроить авторизацию через Telegram

  1. Создайте бота через @BotFather в Telegram
  2. Получите токен и имя бота
  3. Добавьте в .env файл: 
    TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_BOT_USERNAME=имя_бота_без_@
  4. Настройте домен для бота через команду /setdomain в BotFather
  5. Перезапустите сервер: docker-compose restart
Примечание: Для работы на локальном компьютере необходимо настроить HTTPS с валидным сертификатом или использовать сервис типа ngrok для получения публичного HTTPS URL.
